# PropIntel : The Property Intelligence Platform
This is a complete story arc of development of Property Intelligence Platform. it  allows its user to analyse market data in the most visual and costomizable way possible to gain inights about properties that include, Plots, Commercials, Houses/villas, flats.
Data is Generated using Faker by simulating the market synthetically using fake data to simulate the real market trends of property. But the fake data is parametarized by assessing the scraping data from websites or data from apis. 
The cloud provider is aws.
Docker is used to transfer from local instances to cloud instances.
IDE is created and maintained in the VS code editor.
